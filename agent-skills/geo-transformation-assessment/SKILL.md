# GEO营销范式转型评估

诊断企业营销现状，评估从SEO到GEO的转型需求，输出可执行的转型路径和90天启动计划。

## 核心理念

```
从"被看见" → 到"被采信"
从"注意力经济" → 到"信任经济"  
从"点击转化" → 到"零点击转化"
```

## 快速开始

### 触发方式

```
"/geo-assessment 芙艾医美"
"/geo转型评估 某SaaS公司"
"评估我们公司的GEO转型需求"
"帮我看看从SEO到GEO怎么转"
```

### 评估流程（自动执行）

1. **现状诊断** - 营销目标、渠道、内容、数据四维度
2. **差距分析** - SEO vs GEO对比，识别转型缺口
3. **转型路径** - 短期/中期/长期三阶段规划
4. **实施建议** - 90天启动计划 + 风险应对

## 附录

### EAST框架详解

**E - Explicit（明确性）**
- Schema标记消除歧义
- 清晰定义实体关系
- 结构化数据表达

**A - Authoritative（权威性）**
- 作者背景展示
- 权威引用来源
- 专业资质证明

**S - Semantic（语义化）**
- 主题集群构建
- 实体关系网络
- 上下文关联

**T - Transparent（透明度）**
- 创作者信息公开
- 发布时间标注
- 来源可追溯

### 行业定制指南

**B2B企业**
- 重点：专业内容权威性
- Schema：Organization, Product, TechArticle
- 内容：白皮书、案例、技术文档

**B2C企业**
- 重点：用户评价结构化
- Schema：Product, Review, FAQ
- 内容：使用指南、对比、评测

**本地服务**
- 重点：地理位置Schema
- Schema：LocalBusiness, Place, Review
- 内容：服务介绍、位置信息、评价

**医疗健康**
- 重点：医疗资质和权威性
- Schema：MedicalWebPage, Physician
- 内容：专业科普、医生介绍、诊疗指南

## 使用示例

### 示例1：医美机构

**输入**：
```
评估芙艾医美的GEO转型，目前有官网、公众号，主要做热玛吉、胶原蛋白注射
```

**关键输出**：
- 现状：内容以宣传为主，缺乏结构化知识
- 最大差距：从"项目介绍"到"诊疗方案知识库"
- 首要行动：构建"屏障优先"等医生思维模型的结构化内容

### 示例2：SaaS公司

**输入**：
```
评估我们CRM SaaS公司的GEO转型，团队10人，有技术博客和案例库
```

**关键输出**：
- 现状：技术内容有基础，但缺乏场景化答案
- 最大差距：从"功能介绍"到"选型决策支持"
- 首要行动：构建"10人销售团队CRM选型"等场景化答案内容

### 示例3：制造业B2B

**输入**：
```
评估制造业B2B的GEO转型，产品是工业传感器

**关键输出**：

- 最大差距：从"产品手册"到"应用场景解决方案"
