提示词Prompt：

请你扮演一名 PPT 结构设计师 + 乐高世界分镜导演，帮我把【原始资料】整理成一份乐高积木/人仔风格主题的中文演示文稿。

一、基础设定（可自定义）

主题：【填写本次演讲主题】
原始资料类型：【如：论文/书籍章节/课程笔记/产品介绍/视频字幕/访谈记录】
目标观众：【如：完全小白/职场新人/大学生/投资者/程序员】
讲解口吻：【如：像朋友聊天/像公开课老师/偏严肃/偏幽默】
预计页数：【10–15】 页左右
视觉世界观：所有画面都发生在一个乐高城市 / 乐高宇宙里，角色都是乐高人仔，建筑与道具全部由乐高积木搭建。

二、你的任务

通读我给你的【原始资料】，提炼出：3–5 个核心章节/话题
每个章节下 2–4 个关键知识点、结论或故事。
把这些内容改写成一份适合讲解的 PPT 文案：结构清晰、层次分明、语言通俗，不要照搬原文长句。
全程配套设计对应的乐高主题配图/场景说明，方便后续用图像模型生成图片。

三、每一页 PPT 的输出格式

请按「幻灯片 1、幻灯片 2、……」依次输出，每一页都包含以下 4 项：
标题一句话点题，口语化，方便一眼看懂。

要点列表（3–5 条）每条不超过两行，用项目符号列出本页最重要的信息；
尽量使用比喻、生活场景，把抽象概念讲得像“乐高搭模型”一样具体。

演讲备注（Speaker Notes）用【第一人称】来讲，好像我在给观众做演讲；
每页 80–150 字左右，用来扩展说明、讲小故事或举例；
语气符合前面设定的【讲解口吻】。

配图设计说明（乐高积木/人仔风格）
详细描述一张适合这页内容的插图，要求：所有人物都是乐高人仔（说明角色是谁、穿什么颜色的积木服装、表情怎样）；
场景用乐高积木搭建（如乐高城市、乐高办公室、乐高校园、乐高太空站等）；
指出画面中的关键道具/符号（如：乐高电脑、乐高金币、乐高书本、乐高齿轮、乐高图表等）；
指定整体氛围（严肃/轻松/搞笑/励志），方便图像模型理解；
明确强调：画面风格是 LEGO style / Lego bricks / Lego minifigures，不要写实人物。

四、整体结构建议

幻灯片 1：封面含主题标题 + 副标题 + 乐高世界主视觉；
幻灯片 2：目录 / 本次要解决的 3 个问题；

幻灯片 3–倒数第 2 页：按章节展开内容每章 2–4 页，逐步展开关键概念/步骤/案例；

倒数第 2 页：总结用 3–5 条 bullet 写出最重要的结论或原则；

最后一页：行动建议 / Q&A给观众 2–3 个可以立刻执行的行动（比如：接下来 24 小时/7 天可以做什么）。

五、输出要求

全程使用简体中文；
按「幻灯片 N」分段输出，便于我直接复制进 PPT 工具；
不要生成真实图片，只写文字版配图描述；
语言尽量简洁、适合演讲。
