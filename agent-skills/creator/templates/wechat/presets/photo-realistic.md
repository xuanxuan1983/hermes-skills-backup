# Photo-Realistic 写实摄影风

## Color Palette
- `#2D3748` 墨岩灰 — 主色调，沉稳大气的深色基底
- `#4A5568` 钢铁灰 — 辅助色，层次丰富的中间色调
- `#EDF2F7` 云雾白 — 高光色，干净通透的亮部表现
- `#3182CE` 信号蓝 — 强调色，科技感与专业度的视觉锚点

## Illustration Style
写实摄影风格渲染，追求接近真实照片的视觉质感。运用戏剧性光影（侧光、逆光、伦勃朗光）塑造立体感与氛围感。浅景深虚化背景，突出主体。画面构图参考杂志编辑摄影的专业水准，注重黄金比例与视觉引导线。色调偏冷灰，以蓝色系点亮画面。适合商业分析、产品评测、前沿科技、城市生活类文章的配图。
