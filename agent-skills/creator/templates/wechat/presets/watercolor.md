# Watercolor 水彩意境

## Color Palette
- `#B5C7D3` 烟雨蓝 — 主色调，如远山淡影般沉静
- `#E8D5B7` 宣纸黄 — 底色，温润的纸质肌理感
- `#C9B1D0` 紫藤色 — 点睛色，增添文艺与浪漫气息
- `#A8C8A6` 青苔绿 — 辅助色，带来自然清新的生命力

## Illustration Style
水彩画风格，强调颜料在纸面上的自然晕染与渗透效果。色彩层次透明叠加，边缘柔和模糊，呈现有机的流动感。留有适当的"水痕"和"飞白"，营造手绘的温度与呼吸感。构图注重意境留白，以少胜多。适合文化评论、读书笔记、情感散文、文艺话题类文章的配图。
