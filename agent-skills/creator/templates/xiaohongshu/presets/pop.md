# Pop 活力波普

## Color Palette
- `#F56565` 活力红 — 主色，充满能量与热情
- `#ECC94B` 明亮黄 — 高调辅助色，欢快跳跃
- `#4299E1` 电光蓝 — 冷色平衡，增加层次
- `#FF69B4` 霓虹粉 — 点缀色，强化趣味与潮流感

## Typography
粗壮有力的标题字，可带描边或阴影。正文保持清晰可读但风格活泼。数字和关键词使用超大字号+彩色填充。整体排版打破常规网格，倾斜、重叠、大小对比强烈。

## Decorations
对话气泡框、星芒爆炸图形（starburst）、漫画式速度线、粗黑描边、圆点阵列、锯齿形边框、拟声词文字效果（POW/BOOM风格）。高密度视觉元素，画面充实饱满。
