# Cute 甜系少女

## Color Palette
- `#FED7E2` 樱花粉 — 主背景色，柔软如棉花糖
- `#FEEBC8` 蜜桃橘 — 辅助暖调，增添甜蜜气息
- `#C6F6D5` 薄荷奶绿 — 点缀色，平衡甜腻感
- `#FF69B4` 热力粉 — 强调色，用于标题和重点元素

## Typography
圆润可爱的字体风格，笔画末端带有弧度。标题使用加粗圆体，正文保持轻盈的细圆体。数字和英文混排时选用bubble风格字型，整体阅读感受轻松无压力。

## Decorations
爱心、星星闪光、气泡贴纸、蝴蝶结丝带、圆角矩形卡片、波浪形分割线、迷你表情符号点缀。所有边框和容器采用大圆角（16px+），营造膨胀柔软的视觉触感。
