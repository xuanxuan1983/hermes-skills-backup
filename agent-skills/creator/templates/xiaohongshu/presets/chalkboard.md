# Chalkboard 黑板粉笔

## Color Palette
- `#1A1A1A` 黑板色 — 深色背景，模拟真实黑板
- `#F5F5F5` 白粉笔 — 主要书写色，略带粗糙质感
- `#FFE566` 黄粉笔 — 重点标注，最醒目的强调色
- `#FF9999` 粉粉笔 — 次级强调，用于关键词圈注
- `#66B3FF` 蓝粉笔 — 图表和辅助标注

## Typography
模拟手写粉笔字的字体风格，笔画不完全均匀，带有自然的抖动和断笔。标题放大加粗，像老师在黑板上写的重点。可穿插箭头、下划线、圈注等手写标记。

## Decorations
粉笔飞溅粉尘效果、手绘箭头和连线、简笔涂鸦插画、波浪下划线、星号重点标记、方框和圆圈圈注、板擦擦除痕迹。背景添加细微的黑板磨损纹理，增强真实感。
