# Retro 复古潮流

## Color Palette
- `#E07A4D` 陶土橘 — 主色，温暖的复古基调
- `#D4A5A5` 灰玫瑰 — 辅助色，柔和的旧时光气质
- `#6B9999` 褪色青 — 对比色，增添沉稳年代感
- `#F5E6D3` 旧纸黄 — 背景色，泛黄纸页的质感

## Typography
带有年代感的衬线字体或美术字风格，笔画粗细对比明显。标题可模仿旧报刊排印或海报字体。英文部分适合使用Slab Serif或Art Deco风格字型，与中文美术字交相辉映。

## Decorations
网点印刷效果（halftone dots）、复古徽章/邮票框、做旧划痕纹理、圆角矩形标签、胶片边框、老式广告排版元素、磨损边缘效果。色彩整体降低饱和度，模拟褪色感。
