# Minimal 极简高级

## Color Palette
- `#000000` 墨黑 — 标题与核心文字，沉稳有力
- `#FFFFFF` 纯白 — 主背景，极致留白
- `#FAFAFA` 浅灰白 — 卡片或分区底色，微妙层次

可选择性加入单一点缀色（如低饱和蓝或棕），但同时只用一种，保持画面纯净。

## Typography
纤细优雅的无衬线体，字重以Light到Regular为主。标题可使用全大写英文或宽间距中文。行距慷慨，段落间距呼吸充分。所有排版对齐严谨，网格感强。

## Decorations
极细线条分割、大面积留白、单色线性图标、点状网格底纹（极淡）、规则的几何分栏。杜绝多余装饰，每个元素都有明确功能目的。偶尔以单一accent色做克制点睛。
