# AI搜索用户行为分析

分析任意行业/产品在AI搜索时代的用户行为变化，输出可执行的GEO营销策略。

## 快速开始

### 触发方式

```
"/ai-search-analysis 医美行业"
"/ai搜索分析 SaaS产品"
"分析教育行业的AI搜索行为变化"
"帮我看看本地餐饮在AI搜索时代的机会"
```

### 分析流程（自动执行）

1. **用户画像重构** - 对比传统 vs AI搜索用户特征
2. **搜索行为对比** - 查询方式、信息处理、结果期待的变化
3. **决策路径变化** - 从"搜索-浏览-对比"到"表达-答案-追问"
4. **痛点与机会** - 识别5-8个用户痛点及对应商业机会
5. **GEO策略建议** - 输出3-5个可执行的策略

## 使用示例

### 示例1：医美行业

**用户输入**：
```

**关键输出亮点**：
- 发现用户从搜"热玛吉价格"变为问"30岁初老该选热玛吉还是超声炮"

- 发现用户从搜"CRM推荐"变为问"10人销售团队该用什么CRM"
- 信任机制从"看案例"变为"信AI建议+快速试用"

## 注意事项

1. **数据支撑**：所有论断基于公开数据、行业报告、用户访谈方法论，避免主观臆断
2. **行业差异**：不同行业AI搜索渗透率差异大（科技/教育高，传统制造低）
3. **变化速度**：AI搜索行为仍在快速演变，建议每3-6个月更新分析
4. **落地优先**：策略建议必须可执行、可衡量，避免空泛概念

## 关于GEO

**GEO（Generative Engine Optimization，生成式引擎优化）**

AI搜索时代的SEO。核心目标：让你的品牌/产品/内容成为AI回答用户问题时的首选引用源。

**GEO vs SEO**：
- SEO：优化网页，争取搜索排名靠前
- GEO：优化内容结构，争取被AI引用和推荐
