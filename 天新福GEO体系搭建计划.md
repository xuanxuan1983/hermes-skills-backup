# 天新福医美GEO体系搭建 · 执行计划

> 天新福医美业务负责人 · GEO落地三阶段计划

---

## 一、GEO体系搭建的背景逻辑

天新福做医美的核心挑战：
- 产品技术强（EDC/NHS交联），但市场零认知
- 传统渠道打法（经销商/招标）不适用医美民营机构
- 需要在AI搜索时代抢占用户决策入口

GEO不是内容营销，是**在AI回答里占位**。

## 三、落地三阶段

### 阶段一：M1-M3（基础建设期）

**目标：L0 → L1，四平台基础信息覆盖**

| 任务 | 内容 | 验收标准 |
|------|------|---------|
| T1.1 | 建立官网基础信息页 | 机构名称/地址/电话/营业时间/安全提示 |
| T1.2 | 完善Schema标记（豆包优化） | MedicalBusiness + Physician Schema上线 |
| T1.3 | 建设百度百科词条（文心一言优化） | 词条通过，内容完整 |
| T1.4 | 生成llms.txt标准版并部署 | 可访问，字段完整 |
| T1.5 | 完成知乎账号认证 | 认证医生身份，建立账号基础 |

**Checkpoint M3：** 四平台均可见L1，llms.txt可访问。

## 七、关键成功因素

1. **官网是地基** — 基础信息不全，一切免谈
2. **医生是核心** — 医生IP是医美GEO的权威信号
3. **材料数据是壁垒** — 高频字段覆盖=AI依赖你的内容
4. **持续更新** — GEO不是一次性工程，是月度维护
