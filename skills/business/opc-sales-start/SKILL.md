# OPC Sales Start（一人的公司冷启动路径）

## 核心逻辑

**不需要"建好了再帮别人"，帮别人建的过程就是建自己的过程。**

别人要的不是"你有一套完美系统"，别人要的是有人带他开始。

## 冷启动四步

### 第一步：告知身边3个人（Day 1）

发微信告诉3个你觉得可能感兴趣的人：

```
最近我在做一个新项目，叫[你的定位]——帮[目标人群]做[解决的问题]。

本质是[一句话核心价值]。

如果你对[相关话题]有兴趣，我们可以聊聊。有空给我发消息。
```

**关键：不是卖，是告知。不需要完美的话术，先让他们知道。**

### 第二步：有人回复"感兴趣"

当有人回复感兴趣，主动约电话：

```
很高兴你感兴趣！我可以跟你电话聊10-15分钟，了解一下你的情况，看我能不能帮到你。这周你有时间吗？
```

**关键：主动约时间，不要等对方来问"怎么聊"。**

### 第三步：电话诊断框架（不成交）

电话里只做一件事：**了解情况，判断你能不能帮到他**。

```
1. 破冰（2分钟）
   "先简单了解一下你的情况"

2. 三个问题（8分钟）
   - "你现在遇到最头疼的问题是什么？关于X这方面的"
   - "你试过什么方式了吗？效果怎么样？"
   - "如果这个问题解决了，对你来说意味着什么？"

3. 收尾（2分钟）
   - "我大概了解了，你这个情况是XXXX（你的判断）"
   - "我这边能帮到你的主要是XXXX"
   - "你想不想深入聊聊/要不要约下次详细看一下？"
```

**关键：电话里不成交，只做诊断。清晰说出他的问题和你能帮到他的点，对方会主动要下一步。**

### 第四步：第一个客户在帮他的过程中产生

帮别人建的过程 → 你的系统真正跑通 → 第一个付费客户

## 快速行动清单

- [ ] 今天发3条微信给身边3个人
- [ ] 有人回复"感兴趣" → 立刻约电话
- [ ] 电话里只做诊断，不卖
- [ ] 通话后记录对方的核心问题
- [ ] 判断：他是不是你的第一个客户候选人？

## 不要等

- ❌ 等系统建好了再帮别人
- ❌ 等话术完美了再开始
- ❌ 等有成功案例了再卖

- ✅ 先告知、先行动、先在帮别人的过程中完善自己

## 判断标准

你的行动是否有效：**身边3个人知不知道你在做什么**？

- 不知道 → 发那3条微信
- 知道但不感兴趣 → 调整你的告知内容
- 感兴趣 → 约电话 → 诊断
