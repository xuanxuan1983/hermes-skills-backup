# 运镜词汇表

## 基础运镜

| 中文 | 英文 | 效果描述 |
|------|------|---------|
| 推 | Push in / Dolly in | 镜头靠近主体，强调细节 |
| 拉 | Pull out / Dolly out | 镜头远离主体，展现场景 |
| 摇 | Pan | 水平转动，展示环境 |
| 移 | Truck / Dolly | 平行移动，跟随主体 |
| 升 | Crane up | 上升，展现宏大感 |
| 降 | Crane down | 下降，聚焦细节 |
| 跟 | Follow | 跟随主体移动 |
| 绕 | Orbit | 环绕主体，360度展示 |

## 即梦推荐运镜词（中文）

```
镜头缓缓推近
镜头慢慢拉远
镜头从左向右平移
镜头缓慢上升
镜头环绕主体旋转
镜头跟随产品移动
镜头固定，主体入画
镜头微微晃动（手持感）
```

## 复合运镜

```
镜头推近同时微微上升
镜头拉远并环绕旋转
镜头跟随主体同时缓慢上升
```

## 速度控制

| 描述词 | 速度 |
|--------|------|
| 缓缓 / 慢慢 | 慢速 (0.5x) |
| 平稳 | 正常 (1x) |
| 快速 | 快速 (1.5x) |
| 急速 | 极速 (2x) |
