# 第一部分：薪酬方案

## 原则

- 让公司觉得"合理、可量化、有约束"
- 让自己觉得"安全、有奔头"
- 不给"打工感"，给"合伙人感"

---

## 方案一：薪酬结构（推荐）

| 组成部分 | 金额 | 说明 |
|---------|------|------|
| **基础年薪** | 80万元 | 按月发放，13薪 |
| **季度绩效工资** | 12万元（3万/季） | 基础工作达成即发，不与销售挂钩 |
| **年度绩效奖金** | 28万元 | 与年度销售回款目标挂钩 |
| **合计** | **120万元** | |

---

### 年度绩效奖金触发条件

| 指标 | 目标值 | 奖金比例 |
|------|--------|---------|
| M7-M12销售回款 | ≥1500万元 | 14万元 |
| M7-M12战略达成 | ①完成15家A类标杆机构签约②KOL培训≥50人 | 14万元 |
| 两项同时达成 | — | 28万元（全额） |
| 仅达成一项 | — | 14万元（半额） |

→ 逻辑：完成1000万回款拿14万，完成1500万拿28万。跳一跳够得着。

---

## 方案二：进阶激励（合伙人感）

| 组成部分 | 金额/比例 | 说明 |
|---------|---------|------|
| **利润分享** | 医美业务净利润的3% | 按年度结算，与公司共担风险 |
| **战略里程碑奖金** | 单次20-30万元 | 第二款获证 + 1亿年回款达成时各触发一次 |

## 三条底线（绝不松口）

| 红线 | 原因 |
|------|------|
| 年包低于120万 | 低于市场价，且意味着公司不认可你的价值 |
| 试用期薪资打折 | 工作内容一样，不接受打折 |
| 五险一金按最低基数缴 | 这是诚意问题，不是钱的问题 |

# 第二部分：战略规划 · 面试演讲词

## 第一段：为什么是天新福（约1分钟）

> "天新福做医美，最大的差异化不是产品，是团队基因。
>
> 你们的根是手足外科和神经外科——外科出身的医生做注射医美，天然具备三个别人短期追不上的东西：
>
> 第一，解剖精度。外科对层次、间隙、神经血管的理解，远超美容科出身的医生，这是注射安全的基础。
>
> 第二，组织修复视角。瘢痕管理、妊娠纹修复、皮肤重建——这些手足外科最擅长的，和医美的适应症拓展高度重叠。
>
> 第三，学术发表能力。外科发表临床研究比医美容易得多，这个能力将来可以建立竞品无法复制的学术壁垒。
>
> 所以天新福不是'又一家医美胶原品牌'，是'外科组织修复技术的医美转化'。"

> **第一步，建网络。** 2026年9月第一款拿证后，用'牛跟腱+适应症最宽+四规格'三个差异点，在15家头部机构建立标杆。重点开发有学术背景的医生，不是打价格战。

> **第三步，建壁垒。** 利用外科团队的学术能力，在2027-2028年发表瘢痕修复、妊娠纹等适应症的真实世界临床数据，建立其他胶原品牌无法跟随的学术壁垒。

# 第三部分：董事长可能追问的问题

## Q1: "你为什么从斐缦离开？"

> "我在斐缦从500万做到5个亿，用了4年。我在那里的工作已经完成——体系建好了，团队带出来了，赛道打出来了。
>
> 天新福是一个新的机会，从零开始。比起守成，我更擅长打新战场。"

---

## Q2: "120万年包是怎么定的？"

> "两个参考：一是医美行业同级别岗位的市场行情；二是天新福在这个阶段需要的不是执行者，而是能带体系的人。
>
> 我给自己的定价，是基于我能带来的价值，不是基于我上一份工作的收入。"

---

## Q3: "你没做过医疗器械的销售，怎么保证能带好团队？"

> "医美 toB 销售的核心不是医疗器械渠道经验，是三件事：机构关系、学术推广、医生培训。
>
> 这三件事，我过去17年都在做。医疗器械的招标采购流程，我熟。
>
> 但更重要的是——天新福现在需要的不是去跑招标，是帮机构卖货。这件事我做了一辈子。"

---

## Q4: "第二款还没拿证，战略建立在不确定上面，怎么看？"

> "这是我规划里的第一风险。我的应对是：让第一款独立站稳，不依赖第二款。
>
> 第一款拿证后，我有足够的差异化（牛跟腱+四规格+适应症最宽）单独建立市场地位，不需要等第二款背书。
>
> 第二款一旦拿证，疗程体系立刻升级，形成完整闭环。"

---

## Q5: "团队的医美经验不足，怎么补？"

> "这是真实问题。我的策略是：招募有大区销售经验的医美老兵，不要求有胶原经验。
>
> 有医美机构关系的人，三个月内可以上手产品知识。但有胶原经验的人，不一定有机构关系。
>
> 我招有机构关系的人，用体系带产品知识。"

```

面试后：
□ 当场不加微信
□ 当天发一条简短感谢信息（简短，不追问）
□ 等对方主动推进
```

# 第五部分：谈判节奏建议

```
第一轮：董事长出价（让他先说）
  → 听他报什么数字
  → 不当场接受，也不当场拒绝

第二轮：表达意向 + 亮底线
  → "我对这个机会很有诚意，120万是我能接受的底线"
  → 给他算账：我能带来3000万回款，120万是4%的提成比例

第三轮：让步路径（如果对方压价）
  → 底价：100万基础 + 20万绩效（总包120万不变，但降低底薪提高绩效）
  → 绝不松口：试用期打折、五险一金最低基数

如果对方超过120万：
  → 接受，并要求写入合同（绩效触发条件明确写）
  → 多出来的部分可以谈：利润分享比例、团队预算额度
```
