# 竞品数据反推定价法

适用于：医美 toB 产品（植入剂、填充剂、药品、器械）的上市规划阶段。

## 核心逻辑

```
竞品零售价
    ↓ 反推
机构利润（45-55%）+ 经销商利润（15-20%）+ 厂家回款（30-40%）
    ↓
出厂价区间（厂家回款）
    ↓ 设计
天新福定价 = 竞品区间 ± 10-20%（根据差异化程度调整）
```

## 行业利润分配基准

| 环节 | 占比 | 依据 |
|------|------|------|
| 机构利润 | 45-55% | 医美机构需要覆盖场地/人力/营销 |
| 经销商利润 | 15-20% | 渠道商配送+垫资 |
| 厂家回款 | 30-40% | 合理毛利空间 |

**验证方法**：用竞品已知零售价反推，看推算出的厂家回款是否在30-40%区间内。

## 从新氧数据提炼渠道结构

### 关键数据维度

| 数据项 | 分析价值 |
|--------|---------|
| 零售价区间 | 消费者支付意愿，价格带分布 |
| 规格占比（0.5ml vs 1ml）| 主流产品规格，差异化空间 |
| 联合治疗比例 | toC 端疗程组合逻辑 |
| 机构城市分布 | 重点城市优先级 |
| 机构类型（连锁 vs 单体）| 渠道结构判断（注意：平台会掩盖大连锁） |

### 机构类型识别

```python
chain_keywords = ['美莱', '华美', '艺星', '伊美尔', '华韩', '薇琳', '玫瑰']
# 大型连锁在新氧上记录少，不代表没有合作

```

### 注意事项

- **新氧数据 ≠ 全部渠道**：大连锁在线下/私域销售，不依赖新氧
- **一天数据 ≠ 趋势**：需要跨3-6个月数据才能判断季节性/复购率
- **机构数 ≠ 销量**：高频出现机构不等于高销量

## 定价设计流程

```
Step 1: 从竞品零售价反推出厂价区间
  竞品零售价 × 30-40% = 竞品出厂价区间

Step 2: 确定天新福定价位置
  高于竞品（+10-20%）= 高端定位，需要强差异化叙事
  等于竞品 = 正面竞争，需要快速建标杆
  低于竞品（-10-20%）= 入市定位，适合新品牌第一年

Step 3: 设计机构利润空间
  目标：机构利润 ≥ 60%（高于行业均值50%）
  → 这本身就是招商卖点

Step 4: 验证可行性
  机构利润 = (零售价 - 出厂价) / 零售价
  如果低于60%，需要重新审视定价或调整定位
```

## 案例：弗缦新氧数据 → 天新福定价

```
已知：
  弗缦0.5ml 零售均价：2619元
  弗缦1.0ml 零售均价：4375元

反推出厂价（按35%厂家回款）：
  弗缦0.5ml 出厂价 ≈ 2619 × 35% ≈ 917元
  弗缦1.0ml 出厂价 ≈ 4375 × 35% ≈ 1531元

天新福定价建议（对齐高端+差异化）：
  0.5ml 出厂价：880-980元（略低于弗缦，给机构让利）
  0.5ml 零售价：2800-3200元（机构利润65-70%）
  1.0ml 出厂价：1400-1550元
  1.0ml 零售价：4500-5200元
```

## 机构合作模式设计

三种模式对比：

| 模式 | 机构风险 | 厂家风险 | 适用阶段 |
|------|---------|---------|---------|
| 先款后货 | 高（机构观望） | 低 | 市场成熟期 |
| 免费先给 | 低 | 高（资金压力） | 冷启动期 |
| 押金+动销激励 | 低 | 中 | **推荐：新品入市** |

### 推荐：押金+动销激励模式

```
押金体验（第一阶段）：
  首付押金 2000元/规格
  获得：体验装20支 + 培训支持 + 物料包
  义务：60天内动销 ≥ 10支

动销达标 → 正式代理（疗程打包价，8.5折）
动销不达标 → 退回产品，押金退还
```

## 坑与注意事项

1. **不要只看一天数据**：新氧数据有季节性，单日快照无法支撑战略
2. **不要被平台数据骗**：新氧上的大连锁记录少，不代表它们没合作
3. **机构利润空间比出厂价更重要**：机构愿不愿意推，取决于利润，不是出厂价
4. **联合疗程比例是 toC 需求信号**：38%联合HA，说明消费者接受套餐逻辑
